# OpenAPI Documentation

Welcome to the OpenAPI documentation section. This section provides comprehensive API documentation for the INBO Backend API.

## Available Documentation

- [API Reference](/openapi/api-reference) - Complete interactive API reference with all endpoints

## Hosted API Documentation

The API documentation is hosted and available at:

- **[Swagger UI](https://inbo-django-api.azurewebsites.net/api/docs/)** - Interactive API explorer with try-it-out functionality
- **[ReDoc](https://inbo-django-api.azurewebsites.net/api/redoc/)** - Clean, readable API documentation format
- **[OpenAPI Schema](https://inbo-django-api.azurewebsites.net/api/schema/)** - Download the complete OpenAPI 3.0 schema JSON

## Features

- **Interactive Documentation**: Try out API endpoints directly from the documentation
- **Request/Response Examples**: See example requests and responses
- **Schema Definitions**: View detailed schema definitions for all data models
- **Authentication**: Understand how to authenticate and use the API
- **Try It Out**: Test API calls directly from the Swagger UI interface

## API Overview

The INBO Backend API provides endpoints for:

- **Authentication**: OTP-based authentication with JWT tokens
- **User Management**: Profile, inbox, dashboard data
- **Email Operations**: Read, organize, favorite, and manage emails
- **Search**: Full-text search across emails, newsletters, and posts
- **Recommendations**: Personalized content recommendations
- **Subscriptions**: Payment and subscription management
- **Integrations**: Gmail, Pocket, Notion, Evernote
- And much more...

Visit the [API Reference](/openapi/api-reference) page to see the embedded interactive documentation.
