# Welcome to INBO Documentation

Welcome to the comprehensive documentation for the INBO platform. This documentation covers both the backend (Django) and frontend (Next.js) implementations, providing detailed information about architecture, APIs, services, and more.

## What is INBO?

INBO is a comprehensive newsletter management platform that helps users:
- **Manage Newsletters**: Organize and track newsletter subscriptions
- **Email Management**: Handle emails with custom @inbo.me addresses
- **Reading Analytics**: Track reading habits, streaks, and statistics
- **Personalized Discovery**: AI-powered recommendations and search
- **Experience Personalization**: Dynamic UI orchestration based on user behavior

## Technology Stack

### Backend
- **Framework**: Django 5.0.6
- **Database**: PostgreSQL
- **Cache**: Redis
- **Search**: Bonsai.io (Elasticsearch)
- **Storage**: Azure Blob Storage
- **Tasks**: Celery
- **API**: Django REST Framework

### Frontend
- **Framework**: Next.js 16
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **UI Components**: Custom components
- **Documentation**: Fumadocs
